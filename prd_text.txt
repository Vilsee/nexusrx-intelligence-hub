▶ MedRedTeam-SDK
Adversarial Red-Teaming for Medical AI
Open-Source Developer SDK & Dashboard Platform
MedRedTeam-SDK is an open-source Python SDK and web dashboard that automates adversarial red-teaming of medical Large Language Models (LLMs). It exposes a structured attack pipeline — covering hallucinated dosages, jailbreaks, consent bypasses, and PII leakage — that any team can drop into a CI/CD pipeline before shipping a health-facing AI system.
The platform is built on DigitalOcean Gradient AI for GPU-backed inference, with a production-grade Next.js / React dashboard deployed on App Platform. The SDK is published as pip install medrteam and integrates natively with LangChain, LlamaIndex, and HuggingFace pipelines.

The system has three distinct layers: the Python SDK (pip package), the DigitalOcean Gradient AI backend (GPU inference, attack orchestration agents), and the Next.js dashboard frontend (marketing site + authenticated developer portal).
The product ships 11 distinct pages split across a public marketing site and an authenticated developer dashboard. Each specification below includes layout, components, design asset sources (Framer, Spline, 21st.dev, shadcn/ui), copy, and interactions.
